"Bryte Los" (Aplicación Móvil - Hackathon Tecsup) 2022

Se desarrolló una aplicación móvil con Kotlin y MySQL para la detección del Bullying en el Perú.

"Ideal Room" (Aplicación Web - Tecsup) 2023 - 2024

Se desarrolló una aplicación web para simplificar la búsqueda de habitaciones en alquiler para estudiantes de Lima Metropolitana. Utilizó Node.js en el backend, Angular en el frontend, y MongoDB como base de datos.

"Citas Tecsup" (Aplicación Web y Móvil - Hackathon Proyecto Integrador) 2023

Se desarrolló una aplicación móvil y web para la reserva de citas médicas en el Tópico de Tecsup. Se utilizaron tecnologías como Spring Boot, Django, Kotlin y React bajo el enfoque MVC.
